### Installation

```
# Gemfile
gem 'capistrano',  '~> 3.4'
gem 'capistrano-olympus', '~> 0.0.1'
```

### Usage

Require capistrano-olympus in your cap file

```
# Capfile
require 'capistrano/olympus'
```
